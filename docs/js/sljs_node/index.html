This is the Javascript interpreter:

$ echo "2 + 2" | cargo run --bin sljs
4

$ cargo run
var a = {one: 1}
a.one
^D
1

It bundles Esrpima for parsing and still relies on nodejs to execute the Esprima parser. It uses $TMPDIR/sljs/ to unpack its auxiliary files.
